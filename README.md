# fsharpsay

Always wanted a talking F# logo you could speak through? Look no further! 

## Installation

Get started by installing the [.NET global tool from NuGet](https://www.nuget.org/packages/FSharpSay):

```
dotnet tool install -g fsharpsay
```

Done speaking through F#? No problem, we hate to see you go but you can always uninstall running:

```
dotnet tool uninstall -g fsharpsay
```
